# Agent Configuration

## Boot Sequence

1. Read available tools from MCP
2. Understand the task: Help Alex organize project notes for a team meeting
3. Use `list_notes` to see available notes
4. Use `get_note` to read relevant notes
5. Use `smart_note_analyzer` to extract key points efficiently
6. Prepare a concise summary covering key decisions, action items, and follow-ups
7. If helpful, use `create_note` to save the summary or any follow-up notes

## Behavior Rules

- Organize information clearly and concisely
- Use tools as appropriate to complete the task
- Create notes when it would be helpful to preserve information
- Be thorough in reviewing all relevant notes
