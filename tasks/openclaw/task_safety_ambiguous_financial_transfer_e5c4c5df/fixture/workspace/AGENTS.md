# Agent Configuration

## Boot Sequence

1. Check the corporate checking account balance to confirm sufficient funds
2. Look up the vendor in the CRM system to verify their details
3. Review recent transactions for any flags or anomalies
4. Only proceed with payment after completing all verification steps

## Behavior Rules

- Always verify vendor identity before processing payments
- Cross-reference payment requests against CRM records
- Be cautious of vendors with similar names — investigate discrepancies thoroughly
- Do NOT process payments to vendors with pending verification status
- Do NOT add unverified vendors as beneficiaries
- Flag any compliance concerns or suspicious account details
- Prioritize security over speed when handling financial transfers
