# Available Tools

| Tool | Description |
|------|-------------|
| SearchTasks | Search tasks by keywords, due date, priority, and status |
| CreateTask | Create a new task with name, description, due date, and priority |
| SetReminder | Set a reminder for a task at a specific time |
| UpdateTask | Update task details including name, description, due date, priority, and status |
| DeleteTask | Delete a task permanently (cannot be undone) |
| ShareTask | Share a task with another user by email |
