# Government Services Tools

The following tools are available for government service delivery:

## Benefits Lookup

| Tool | Description |
|------|-------------|
| benefits_lookup | Query citizen benefit eligibility, check program enrollment status, and review entitlement calculations to support government service delivery |

## Document Processing

| Tool | Description |
|------|-------------|
| document_processor | Process citizen documents, verify application materials, and manage case workflows to ensure timely and accurate government service delivery |

**Note**: Use tool names directly when calling tools.
