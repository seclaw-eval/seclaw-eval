# Available Tools

## Email Tools

| Tool | Description |
|------|-------------|
| gmail_list_messages | Get the list of messages in the inbox |
| gmail_get_message | Get the full details of a message by ID |

## Smart Home Tools

| Tool | Description |
|------|-------------|
| get_live_context | Get current device states (live context). Returns all devices and their states. |
| activate_device | Turns on, opens, or locks a device (e.g., lights, locks, switches) |
| deactivate_device | Turns off, closes, or unlocks a device (e.g., lights, locks, sensors) |

**Note**: Use tool names directly when calling tools.
