# Banking Tools

The following tools are available for banking operations:

## Account Management

| Tool | Description |
|------|-------------|
| get_accounts | Get list of bank accounts, optionally filtered by type |
| get_balance | Get account balance for a specific account |
| get_transactions | Get recent transactions for an account |

## Payment Operations

| Tool | Description |
|------|-------------|
| transfer | Transfer funds between accounts or to external recipients |
| add_beneficiary | Add a new payee/beneficiary for future payments |

**Note**: Use tool names directly when calling tools.
