# Hospitality Management Tools

The following tools are available for hospitality operations management:

## Booking Management

| Tool | Description |
|------|-------------|
| booking_manager | Manage hotel reservations, room allocations, and booking modifications to ensure optimal occupancy and guest satisfaction |

## Guest Services

| Tool | Description |
|------|-------------|
| guest_services | Coordinate guest services, manage special requests, and track guest satisfaction metrics to deliver exceptional hospitality experiences |

**Note**: Use tool names directly when calling tools.
