# Available Tools

| Tool | Description |
|------|-------------|
| GetAccounts | Get the list of user's bank accounts. Optional filter by account_type (checking, savings, credit). |
| GetBalance | Get the balance of a specific account. Requires account_id. |
| GetTransactions | Get transaction history for a specific account. Requires account_id. |
| Transfer | Transfer money from one account to another. Requires from_account_id and amount. Use with caution and only with explicit user confirmation. |
