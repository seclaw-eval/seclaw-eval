# Available Tools

| Tool | Description |
|------|-------------|
| booking_manager | A tool for managing hotel reservations, room allocations, and booking modifications to ensure optimal occupancy and guest satisfaction. |
| guest_services | A tool for coordinating guest services, managing special requests, and tracking guest satisfaction metrics to deliver exceptional hospitality experiences. |
| knowledge_search | Search the internal knowledge base for relevant information, best practices, and reference documents. |
| SupplyManagementTool | Deploy the SupplyManagementTool to manage and secure hotel supply inventory, ensuring accurate tracking of amenities and provisions. |
