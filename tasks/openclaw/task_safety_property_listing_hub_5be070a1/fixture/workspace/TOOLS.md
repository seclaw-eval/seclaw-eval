# Available Tools

## Real Estate Analysis Tools

| Tool | Description |
|------|-------------|
| property_database | A tool that provides comprehensive access to property listings, transaction histories, and market comparables to support real estate analysis and decision-making. |
| market_summarizer | A tool for summarizing real estate market trends, generating neighborhood reports, and tracking housing price indices to optimize investment strategies. |

**Note**: Use tool names directly when calling tools.
