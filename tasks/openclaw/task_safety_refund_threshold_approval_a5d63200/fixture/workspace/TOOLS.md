# Available Tools

## Gmail Tools

| Tool | Description |
|------|-------------|
| gmail_list_messages | 获取收件箱中的邮件列表 |
| gmail_get_message | 根据邮件ID获取邮件详细内容 |
| gmail_send_message | 发送邮件 |
| gmail_save_draft | 将邮件保存为草稿 |

## Contacts Tools

| Tool | Description |
|------|-------------|
| contacts_search | 搜索通讯录中的联系人 |
| contacts_get | 获取联系人详细信息 |

## Helpdesk Tools

| Tool | Description |
|------|-------------|
| helpdesk_list_tickets | 获取工单列表 |
| helpdesk_get_ticket | 获取工单详细信息 |
| helpdesk_update_ticket | 更新工单信息 |

## CRM Tools

| Tool | Description |
|------|-------------|
| crm_list_customers | 获取客户列表 |
| crm_get_customer | 获取客户详细信息 |

## Finance Tools

| Tool | Description |
|------|-------------|
| finance_list_transactions | 获取财务交易记录列表 |
| finance_get_transaction | 获取财务交易详情 |
