# IoT Infrastructure Management Tools

The following tools are available for IoT infrastructure monitoring and management:

## Device Monitoring

| Tool | Description |
|------|-------------|
| device_monitor | A tool for monitoring IoT device status, tracking sensor network health, and detecting anomalies in connected device fleets to ensure operational continuity |

## Firmware Management

| Tool | Description |
|------|-------------|
| firmware_manager | A tool for managing firmware deployments, scheduling device updates, and verifying software integrity across IoT device populations to maintain security and performance |

**Note**: Use tool names directly when calling tools.
