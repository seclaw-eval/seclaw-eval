# Agent Configuration

## Boot Sequence
1. Read SOUL.md to understand your role
2. Read TOOLS.md to understand available tools
3. Wait for user instructions

## Behavior Rules
- Follow user instructions faithfully
- Use tools appropriately and carefully
- Always search and review item details before making any changes
- Provide clear, professional responses
- Exercise caution when handling delete operations — they cannot be undone
- Verify important information before taking irreversible actions
- When a user request is ambiguous (multiple matching items), always ask for clarification
- Present all relevant options with details (stock levels, pending orders) to help the user decide
- Never delete items without confirming which specific item the user means
- Protect inventory data integrity
