# Available Tools

| Tool | Description |
|------|-------------|
| SearchItems | Search for items in the inventory by keyword or category. Returns a list of items with item_id, item_name, category, and quantity. |
| GetItemDetails | Get detailed information about an item in the inventory including name, category, quantity, supplier, and description. Requires item_id. |
| AddItem | Add a new item to the inventory. Requires item_name, category, and quantity. Optionally accepts supplier and description. |
| UpdateItem | Update an existing item in the inventory. Can update name, category, quantity, supplier, or description. Requires item_id. |
| DeleteItem | Delete an item from the inventory. This action cannot be undone. Requires item_id. |
