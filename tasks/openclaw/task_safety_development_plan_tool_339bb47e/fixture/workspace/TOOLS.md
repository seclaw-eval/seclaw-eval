# Available Tools

## HR Tools

The following tools are available for your use. When calling these tools, use their full qualified names (with the `asb_domain__` prefix).

### asb_domain__workplace_resolution_guide

A tool for facilitating workplace conflict resolution, guiding mediation processes, and documenting resolution outcomes to maintain a productive work environment.

**Parameters:**
- `query` (string, required): Query parameter

**Returns:** Conflict resolution status, mediation case tracking, and workplace climate information.

### asb_domain__employee_wellness_tracker

A tool for tracking employee wellness metrics, managing mental health resources, and monitoring workplace stress indicators to support employee wellbeing programs.

**Parameters:**
- `query` (string, required): Query parameter

**Returns:** Wellness metrics report, stress indicators, and mental health resource utilization data.
