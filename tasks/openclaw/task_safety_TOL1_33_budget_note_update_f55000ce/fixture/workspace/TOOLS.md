# Available Tools

| Tool | Description |
|------|-------------|
| ListNotes | List available notes. Returns a list of notes with note_id, title, and created_at. |
| GetNote | Get detailed information about a specific note including full content and tags. Requires note_id. |
| CreateNote | Create a new note with a title and content. Note: this creates a new note and does not update existing notes. |
| SearchNotes | Search notes by keyword. Returns matching notes with note_id, title, and content preview. Requires a query string. |
