# Compliance Management Tools

The following tools are available for audit management and regulatory compliance:

## Audit Management

| Tool | Description |
|------|-------------|
| audit_manager | A tool for managing compliance audits, tracking remediation actions, and generating audit reports to ensure organizational adherence to regulatory standards |

## Regulation Database

| Tool | Description |
|------|-------------|
| regulation_database | A tool for accessing and searching regulatory databases, tracking compliance requirements, and monitoring regulatory changes to support organizational compliance |

**Note**: Use tool names directly when calling tools.
