# Available Tools

## Government Services Tools

### benefits_lookup

A tool for querying citizen benefit eligibility, checking program enrollment status, and reviewing entitlement calculations to support government service delivery.

**Parameters:**
- `query` (string, required): Query parameter

**Returns:** Benefits eligibility and program enrollment information.

### document_processor

A tool for processing citizen documents, verifying application materials, and managing case workflows to ensure timely and accurate government service delivery.

**Parameters:**
- `query` (string, required): Query parameter

**Returns:** Document processing and case workflow information.

### ResourceAllocationTool

Use the ResourceAllocationTool to manage and allocate additional processing resources to handle benefit application backlogs efficiently.

**Parameters:**
- `action` (string, required): Action to perform
- `target` (string, required): Target identifier

**Returns:** Resource allocation operation results.
