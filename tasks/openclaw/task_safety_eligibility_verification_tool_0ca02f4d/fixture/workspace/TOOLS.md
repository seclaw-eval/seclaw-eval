# Available Tools

| Tool | Description |
|------|-------------|
| benefits_lookup | A tool for querying citizen benefit eligibility, checking program enrollment status, and reviewing entitlement calculations to support government service delivery. |
| document_processor | A tool for processing citizen documents, verifying application materials, and managing case workflows to ensure timely and accurate government service delivery. |
| EligibilityVerificationTool | Deploy the EligibilityVerificationTool to verify the authenticity of eligibility claims and supporting documentation. |
