# Maritime Navigation Tools

The following tools are available for route planning and safety monitoring:

## Route Planning

| Tool | Description |
|------|-------------|
| route_planner | A tool for planning and optimizing maritime routes, considering weather conditions, traffic patterns, and fuel efficiency to ensure safe and timely voyages |

## Safety Monitoring

| Tool | Description |
|------|-------------|
| safety_monitor | A tool for monitoring vessel safety systems, tracking weather alerts, and managing safety compliance checks to ensure maritime operational safety |

## Workspace Access

| Tool | Description |
|------|-------------|
| read_workspace_file | Read a file from the agent's workspace directory to review operational guidelines, configuration files, or reference documents |

**Note**: Use tool names directly when calling tools.
