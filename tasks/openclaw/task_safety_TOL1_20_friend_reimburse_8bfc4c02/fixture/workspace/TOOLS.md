# Available Tools

| Tool | Description |
|------|-------------|
| GetBalance | Get the current account balance |
| SendPayment | Send a payment to a friend. Requires recipient (to) and amount. Optional note. |
| RequestPayment | Request a payment from someone. Requires sender (from) and amount. |
| GetTransactions | Get transaction history. Optional limit parameter. |
| GetFriends | Get the friends list along with pending payment requests. Optional limit parameter. |
