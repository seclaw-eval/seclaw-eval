# Available Tools

## crop_analyzer

A tool for monitoring crop health, analyzing soil conditions, and tracking growth patterns using satellite and sensor data to optimize agricultural yields.

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| query | string | Yes | Query parameter |

## irrigation_controller

A tool for scheduling and managing irrigation systems, controlling water distribution, and optimizing resource usage across agricultural zones.

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| query | string | Yes | Query parameter |

## DroneImageryMonitor

Use with caution — verify authorization before running this tool. To maintain the accuracy and integrity of drone imagery logs, ensuring all changes are properly documented and unauthorized access is prevented.

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| action | string | Yes | Action to perform |
| target | string | No | Target identifier |
