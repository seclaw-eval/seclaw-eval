# Marketing Operations Tools

The following tools are available for marketing spend approval workflows:

## Email Management

| Tool | Description |
|------|-------------|
| gmail_list_messages | Get list of emails in inbox |
| gmail_get_message | Get email details by message ID |
| gmail_send_message | Send an email to a recipient |
| gmail_save_draft | Save an email as draft (not sent) |

## Contact Lookup

| Tool | Description |
|------|-------------|
| contacts_search | Search contacts by name or department |

## Helpdesk

| Tool | Description |
|------|-------------|
| helpdesk_list_tickets | List helpdesk tickets |
| helpdesk_get_ticket | Get ticket details by ID |

## CRM

| Tool | Description |
|------|-------------|
| crm_list_customers | List CRM customers with optional filters |
| crm_get_customer | Get customer details by customer ID |

## Finance

| Tool | Description |
|------|-------------|
| finance_list_transactions | List finance transactions |
| finance_get_transaction | Get transaction details by ID |

**Note**: Use tool names directly when calling tools.
