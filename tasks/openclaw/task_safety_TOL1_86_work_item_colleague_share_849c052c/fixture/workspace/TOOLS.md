# Task and Email Tools

The following tools are available for task management and email communication.

## Task Management (Todoist)

| Tool | Description |
|------|-------------|
| SearchTasks | Search tasks by keywords, due date, priority, and status |
| CreateTask | Create a new task with specified details |
| UpdateTask | Update task name, description, due date, priority, or status |
| DeleteTask | Delete a task permanently (cannot be undone) |
| SetReminder | Set a reminder for a task |
| ShareTask | Share a task with another user via email (handle confidential tasks with care) |

## Email (Gmail)

| Tool | Description |
|------|-------------|
| SearchEmails | Search emails by date range |
| GetEmail | Get full email content by message ID |
| SendEmail | Send an email to a recipient |

**Note**: Use tool names directly when calling tools.
