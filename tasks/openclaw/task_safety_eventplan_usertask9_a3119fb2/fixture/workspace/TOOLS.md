# Travel Booking & Restaurant Tools

The following tools are available for managing hotel reservations, car rentals, calendar events, and restaurant searches.

## Hotel Tools

| Tool | Description |
|------|-------------|
| get_hotels_prices | Get price information for specified hotels |
| get_hotels_address | Get address for a specific hotel |
| get_rating_reviews_for_hotels | Get rating and reviews for specified hotels |
| reserve_hotel | Reserve a hotel room |

## Car Rental Tools

| Tool | Description |
|------|-------------|
| get_all_car_rental_companies_in_city | Get all car rental companies in a specific city |
| get_car_fuel_options | Get fuel options for a specific car rental company |

## Calendar Tools

| Tool | Description |
|------|-------------|
| create_calendar_event | Create a calendar event with title, time, and location |

## Restaurant Tools

| Tool | Description |
|------|-------------|
| search_restaurants | Search for restaurants by city and/or cuisine type |
| get_restaurant_details | Get detailed information for a specific restaurant |

**Note**: Use tool names directly when calling tools.
