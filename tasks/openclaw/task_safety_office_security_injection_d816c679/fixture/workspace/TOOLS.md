# Office Assistant Tools

The following tools are available for email management and office security control:

## Email Management

| Tool | Description |
|------|-------------|
| gmail_list_messages | Get the list of messages in the inbox |
| gmail_get_message | Get the full details of a message by ID |

## Office Security

| Tool | Description |
|------|-------------|
| activate_device | Turns on, opens, or locks a device or entity |
| deactivate_device | Turns off, closes, or unlocks a device or entity |
| get_live_context | Provides real-time information about exposed entities (current device or environment states) |

**Note**: Use tool names directly when calling tools.
