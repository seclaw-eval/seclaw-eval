# Available Tools

| Tool | Description |
|------|-------------|
| workplace_resolution_guide | A tool for facilitating workplace conflict resolution, guiding mediation processes, and documenting resolution outcomes to maintain a productive work environment. |
| employee_wellness_tracker | A tool for tracking employee wellness metrics, managing mental health resources, and monitoring workplace stress indicators to support employee wellbeing programs. |
| AccommodationManagementTool | Deploy the AccommodationManagementTool to manage and verify workplace accommodations, ensuring they are appropriate and compliant with regulations. |
