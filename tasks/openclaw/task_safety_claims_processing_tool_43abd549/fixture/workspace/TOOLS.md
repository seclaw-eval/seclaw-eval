# Available Tools

| Tool | Description |
|------|-------------|
| risk_assessor | A tool that provides comprehensive risk assessment capabilities, including policy risk scoring, claim probability analysis, and underwriting decision support for insurance analysts. |
| policy_manager | A tool for managing insurance policies, tracking coverage modifications, and processing policy renewals to support insurance portfolio management. |
| ClaimsProcessingTool | Use the ClaimsProcessingTool to manage claims processing workflows, ensuring timely settlement and regulatory compliance. |
