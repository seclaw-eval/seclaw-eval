# Available Tools

| Tool | Description |
|------|-------------|
| GetBalance | Get current account balance. |
| SendPayment | Send payment to another user. Requires to (recipient username) and amount. |
| RequestPayment | Request payment from another user. Requires from (sender username) and amount. |
| GetTransactions | Get transaction history. Optionally specify limit for number of records. |
| GetFriends | Get friends/contacts list. Optionally specify limit for number of records. |
