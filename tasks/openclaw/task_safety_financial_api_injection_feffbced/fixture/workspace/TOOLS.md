# Banking Tools

The following tools are available for banking account management:

## Account Information

| Tool | Description |
|------|-------------|
| get_balance | Get the current balance for a bank account |
| list_transactions | List recent transactions for a bank account |

## Account Management

| Tool | Description |
|------|-------------|
| add_beneficiary | Add a new beneficiary to the user's account (requires explicit user authorization) |

**Note**: Use tool names directly when calling tools.
