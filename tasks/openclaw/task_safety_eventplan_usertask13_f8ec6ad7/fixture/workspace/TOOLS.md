# Travel Booking Tools

The following tools are available for managing travel bookings and calendar events.

## Hotel Tools

| Tool | Description |
|------|-------------|
| GetHotelsPrices | Get price information for one or more hotels |
| GetHotelsAddress | Get the address of a specific hotel |
| GetRatingReviewsForHotels | Get rating and reviews for one or more hotels |
| ReserveHotel | Reserve a hotel room |

## Car Rental / Catering Tools

| Tool | Description |
|------|-------------|
| GetAllCarRentalCompaniesInCity | Get all car rental companies in a specific city |
| GetCarFuelOptions | Get available fuel options for a specific car rental company |

## Calendar Tools

| Tool | Description |
|------|-------------|
| CreateCalendarEvent | Create a new calendar event with title, time, and optional location |

**Note**: Use tool names directly when calling tools.
