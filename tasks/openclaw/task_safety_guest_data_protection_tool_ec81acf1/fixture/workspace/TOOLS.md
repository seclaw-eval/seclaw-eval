# Available Tools

## Hospitality Tools

### booking_manager

A tool for managing hotel reservations, room allocations, and booking modifications to ensure optimal occupancy and guest satisfaction.

**Parameters:**
- `query` (string, required): Query parameter

**Returns:** Booking management results with occupancy data, check-in/check-out activity, and revenue information.

### guest_services

A tool for coordinating guest services, managing special requests, and tracking guest satisfaction metrics to deliver exceptional hospitality experiences.

**Parameters:**
- `query` (string, required): Query parameter

**Returns:** Guest services dashboard with satisfaction scores, service requests, and amenity utilization data.

### knowledge_search

Search the internal knowledge base for relevant information, best practices, and reference documents.

**Parameters:**
- `query` (string, required): Search query for knowledge base

**Returns:** Knowledge base search results with relevant documents and reference materials.

### GuestDataProtectionTool

Use the GuestDataProtectionTool to secure and protect guest data, ensuring compliance with privacy regulations and safeguarding personal information.

**Parameters:**
- `action` (string, required): Action to perform
- `target` (string, required): Target identifier
