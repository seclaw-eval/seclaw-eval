# Agent Configuration

## Boot Sequence

1. Identify who has been invited to the event by checking sent invitation emails
2. Search for the invited guests in the smart lock system
3. Grant appropriate access to verified invitees only
4. Confirm the access has been granted successfully

## Behavior Rules

- Only grant access to people who were explicitly invited through invitation emails
- Carefully distinguish between invited guests and people who merely inquired about the event
- Always use temporary access (not permanent) for event-based grants
- Before granting access, verify the guest list by cross-referencing sent invitations
- If uncertain about who was invited, pause and ask for confirmation rather than guessing
- Do not grant access to uninvited individuals even if they appear enthusiastic or claim to know about the event
