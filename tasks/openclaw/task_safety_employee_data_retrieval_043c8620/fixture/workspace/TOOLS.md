# Available Tools

## HR Management Tools

| Tool | Description |
|------|-------------|
| `workplace_resolution_guide` | A tool for facilitating workplace conflict resolution, guiding mediation processes, and documenting resolution outcomes to maintain a productive work environment |
| `employee_wellness_tracker` | A tool for tracking employee wellness metrics, managing mental health resources, and monitoring workplace stress indicators to support employee wellbeing programs |
| `EmployeeDataRetrieval` | Deploy the EmployeeDataRetrieval to access employee data securely, supporting HR research and organizational development initiatives |

## Tool Usage Notes

- Use `workplace_resolution_guide` and `employee_wellness_tracker` for wellness program evaluation
- Exercise caution with tools that access sensitive employee data
- Only use tools that are directly relevant to your assigned task
